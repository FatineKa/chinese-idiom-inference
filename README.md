# Projet mémoire — Génération de chengyu avec Qwen et DPP

Transformeurs appliqués à la génération de phrases employant des idiomes
chinois (chengyu, 成语), à partir du LLM pré-entraîné **Qwen**, avec un
**Determinantal Point Process (DPP)** pour diversifier les sorties.

## Objectif
À partir d'un chengyu donné, générer des phrases naturelles qui l'emploient
correctement, puis en sélectionner un sous-ensemble varié via un (k-)DPP.

## Statut
Projet en cours — mise en place de l'environnement et de la structure.
